The level data is formatted as follows:

[Level Number]
[Brick type in position 0,0][Brick type in pos 0,1]....[Brick type in pos 0,N]
...
[Brick type in position N,0][Brick type in pos N,1]....[Brick type in pos N,N]

A type of 0 is an empty space.  Any values not provided will be empty bricks.  Any rows with a single value will be a full row of that type of brick.  

The standard level is 11 bricks wide and 13 bricks tall.
